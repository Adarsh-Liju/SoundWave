# SoundWave

A music player using `SFML` library

## Dependencies

For Debian/Ubuntu systems SFML can be installed using

```bash
sudo apt-get install libsfml-dev
```

## Build

```bash
make
```

## Usage

```bash
./music_player <name_of_the_music>
```
